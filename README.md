# ai-news-sentiment-backtester
 발표자료
- Github 레포지토리 (팀원간 협업  필수)
- Github 레포지토리는 반드시 Public(전체공개)으로 설정
- README.md에 팀원별 역할 및 기여도 명시
- 팀원 간 협업 확인을 위해 각자 브랜치를 파서 작업해야 하며, 최종 코드는 반드시 깃허브 PR(Pull Request) 승인을 거쳐 main에 머지
- 실제로 돌아가는 모습을 README.md에 스크린샷이나 1분 미만의 GIF(움짤)/동영상 링크로 첨부
- 프로젝트 코드
- 프롬프트 기록 (발표자료와 깃허브 주소가 적힌 텍스트만 제출하고, 코드와 프롬프트 기록은 깃허브 레포지토리 안에 포함시킬 것)
