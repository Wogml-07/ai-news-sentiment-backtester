#  AI 기반 뉴스 감성 분석 백테스터 (ai-news-sentiment-backtester)

> **인공지능개론 기말 팀 프로젝트**
> 실시간 뉴스 데이터와 NLP(자연어 처리) 모델을 결합한 투자 심리 분석 및 백테스팅 시스템

---

## 프로젝트 제출 및 협업 규칙
* 본 레포지토리는 반드시 Public(전체공개)으로 유지
* 팀원 간 협업 및 추적을 위해 각자 기능별 브랜치(Branch)에서 작업하며, 최종 코드는 반드시 PR(Pull Request) 승인을 거쳐 `main` 브랜치에 머지합니다.
* 본 레포지토리 내에 프로젝트 코드와 AI 프롬프트 기록을 모두 포함하여 제출합니다.
* 프로그램이 실제로 구동되는 모습을 담은 스크린샷 또는 1분 미만의 GIF(움짤)/동영상 링크를 하단에 첨부합니다.
